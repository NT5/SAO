setTemplate()

setTemplate(string  $template)

Establece el archivo que será usado de plantilla en el objeto actual.

El Archivo tiene que estar ubicado en $Twig_Template_Folder. Solo se puede cargar un archivo por objeto, si quieres cargar más de uno tiene que usar la sintaxis Twig

Parameters

string $template

Nombre del archivo o ubicación relativa a $Twig_Template_Folder

Examples

Establecer archivo de plantilla
> $Twig->setTemplate($template);

$Twig = new \SAO\Modules\WebPage\Twig();

**Carga de plantilla Twig en directorio raíz**
$Twig->setTemplate('home.twig');

**Carga de plantilla Twig en subdirectorios**
$Twig->setTemplate('pages/home.twig');

addFilter()

addFilter(string  $filter_name, string  $filter_function)

Añade filtros del módulo Twig

Parameters

string $filter_name

Nombre del filtro que sera llamado desde las plantillas Twig
string $filter_function

Funcion PHP que sera invocada en las plantillas Twig

Examples

Establece un filtro nuevo
> $Twig->addFilter($filter_name, $filter_function);

**Filtro Personalizado de la función date**

    // do something different from the built-in date filter
}));

setVar()

setVar(string  $name, string|array  $value)

Agrega variables que serán usadas posteriormente en la plantilla Twig

Parameters

string $name

Nombre de la variable (Pueden ser multidimensionales separadas por un punto)
string|array $value

Valor de la variable puede ser un array o cualquier tipo de objeto

Examples

Añade variables
> $Twig->setVar($name, $value);

**Agrega variables al modulo**
$Twig->setVar('foo', 'bar');

**Agrega variable multidimensional**
$Twig->setVar('foo.bar', 'foo');

**Agrega un arreglo como variable**
$Twig->setVar('foo', array('foo' => 'bar'));
